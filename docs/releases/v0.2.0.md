# LittleClaw Companion v0.2.0

Unified Canvas release for LittleClaw Companion.

## Highlights

- Unified Canvas renderer across all supported species and stages
- Stronger pet-first desktop layout with redesigned compact badge and expanded stage
- Added new species lines: moth, slime, and avian
- Added role-aware pet identity and voice cues
- Improved learn / send / reply flow integration with current OpenClaw sessions
- Added local debug tooling for ongoing development while keeping release builds hideable

## Included

- installer-managed runtime
- pet service
- Companion desktop UI
- preset-driven species, rarity, role, and evolution rules
- release package for local installation and testing

## Install

```bash
./installer/install.sh
```

## Upgrade

```bash
./installer/upgrade.sh
```

## Uninstall

```bash
./installer/uninstall.sh
```

## Release Asset

Recommended local test package:

- `dist/littleclaw-companion-release.tar.gz`

## Notes

- Existing pet progress is preserved by default.
- Care actions now have stronger pet feedback, while real OpenClaw work still drives XP and evolution.
- Local debug tools remain in source for development, but release packaging can hide the debug entry.
