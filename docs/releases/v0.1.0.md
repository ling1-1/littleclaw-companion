# LittleClaw Companion v0.1.0

First public release of LittleClaw Companion.

## Highlights

- Floating desktop companion for OpenClaw
- Real pet progression driven by actual OpenClaw usage
- XP + affinity split growth system
- Hunger / energy / sleep constraints tied to real behavior
- Screenshot, file, and direct-send bridge into the current OpenClaw session
- Installer-managed runtime, presets, assets, and LaunchAgents
- First-pet onboarding flow
- Species / rarity preset foundation for future multi-pet expansion

## Included

- installer-managed runtime
- pet service
- Companion desktop UI
- preset-driven pet rules
- release test package

## Install

```bash
./installer/install.sh
```

## Upgrade

```bash
./installer/upgrade.sh
```

## Uninstall

```bash
./installer/uninstall.sh
```

## Release Asset

Recommended local test package:

- `dist/littleclaw-companion-release.tar.gz`

## Notes

- Existing pet progress is preserved by default.
- Care actions mainly restore state and affinity; real OpenClaw work drives XP and evolution.
- This release is the first public baseline and will continue evolving toward a reusable plugin-style package.
