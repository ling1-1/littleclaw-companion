# LittleClaw Companion v0.3.0

Pet-first Canvas expansion release for LittleClaw Companion.

![Current LittleClaw Companion UI](../assets/current-ui-design.png)

## Highlights

- Expanded the unified Canvas renderer into more species lines, including fox, beetle, and koi
- Reworked compact badge, pet island, onboarding, and encounter flows around a stronger pet-first desktop layout
- Synced the island state more closely with real OpenClaw session activity
- Improved first-pet onboarding, encounter modal visibility, and single-instance runtime behavior
- Kept local debug capability in source while hiding the debug entry in normal UI

## Included

- installer-managed runtime
- pet service
- Companion desktop UI
- preset-driven species, rarity, role, and evolution rules
- release package for local installation and testing

## Notes

- Existing pet progress is preserved by default.
- Care actions still focus on state recovery and affinity; real OpenClaw work remains the main XP and evolution driver.
- Release packaging keeps the debug entry hidden by default while local development tooling stays available in source.
