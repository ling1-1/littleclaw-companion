# LittleClaw Companion v0.3.1

Stability and release-hardening update for the pet-first Canvas Companion.

![Current LittleClaw Companion UI](../assets/current-ui-design.png)

## 中文说明

这一版不是重新推翻 UI，而是在当前宠物主导的 Canvas 架构上继续收口：

- 保留统一 Canvas 宠物渲染和灵动岛工作台
- 继续保留真实学习 / 发送 / 截图 / 文件 / 回执链路
- 修复输入过程中容易被状态同步打断的问题
- 补齐 release 文档、工程沉淀和发布检查清单

## Highlights

- Improved Chinese input stability by deferring host-side `pushState()` updates while text input is active
- Unified staged screenshot / image / file summaries across learning, send, screenshot, and file-pick flows
- Refined release documentation, engineering notes, and smoke-test checklist around the current UI
- Kept debug tooling in source while confirming release packaging still hides the debug entry by default

## Included

- installer-managed runtime
- pet service
- Companion desktop UI
- preset-driven species, rarity, role, and evolution rules
- release package for local installation and testing
